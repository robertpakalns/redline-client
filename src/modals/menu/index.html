Resource Swapper
To use the swapper:
  1. Make sure you enabled it in the menu modal Settings section
  2. Get the file URL you want to swap and extract only the file name and extension
  3. Open the swapper folder and place the file there
  4. Reload the client to apply changes
  5. Example with Voxtulate Client (Extended Swapper)
Changelog
Redline Client

Redline Client is an unofficial Kirka.io client.

Features

  • Adblocker
  • Changelog
  • Client and Engine Versions in-game
  • Customizations
    • Custom Keybinding
    • Fast CSS
    • Userscripts
    • Userstyles
    • Resource Swapper
  • Deeplinks (redline: Protocol)
  • Discord Rich Presence
  • Menu Modal (F1)
  • Proxy Domain Support
  • Tricko Links in Player Modals

This project is still raw; more features will be added soon!

Client Safety
This project is open-source. All Redline Client source code is publicly available in the GitHub repository. All builds are executed via GitHub Actions. If you have concerns about the safety of your private information while using Redline Client, feel free to inspect the source code. Trust in the client is based on trust in the developer.
